# Pioneer

## Purpose

Container for category-defining pioneer projects.

## Why This Folder Exists

Studies of projects that created or first defined a category, used as reference analogs. It is intentionally empty until validated artifacts, produced through the templates and pipeline, are added.

## Source of Truth

Validated research artifacts produced downstream; empty for now.

## Input

Completed template-based knowledge artifacts.

## Output

Reference artifacts for framework validation.

## Consumer

Reviewers and the reasoning-validation process.
