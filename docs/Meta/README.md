# Meta

## Purpose

Documents the **market meta-layer**: narratives, market cycles, hype, trends, and capital rotation that surround projects and shape their reception.

## Why This Folder Exists

A project's fate is heavily influenced by market context. This folder models that context as structured dimensions so reasoning can account for it.

## Source of Truth

Research on market narratives, cycles, and attention.

## Input

Narrative and market-context research.

## Output

Structured definitions of narrative/market meta-attributes.

## Consumer

Reasoning, Patterns, and Valuation layers.

## Folder Structure

```
Meta/
├── Narratives.md
├── MarketCycles.md
├── Hype.md
├── Trend.md
└── Rotation.md
```

## Workflow Position

Context layer feeding the `Ontology`→`Patterns`→`Reasoning` stages of `Research → Knowledge → Ontology → Patterns → Reasoning → Framework → Applications`.

## Rules

1. Model concepts, not live market predictions.
2. Keep fields measurable/classifiable.
3. No dated data.

## Naming Convention

`PascalCase.md`.

## Future Expansion

Real-time sentiment and attention indicators.
