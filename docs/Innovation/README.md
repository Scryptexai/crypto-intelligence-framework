# Innovation

## Purpose

Documents the **innovation-assessment dimensions**: first-mover, fast-follower, copycat, category-creator, competitive moat, product-market fit, timing, execution, and an aggregate innovation score.

## Why This Folder Exists

Innovation is a major driver of long-term outcomes. This folder defines the reusable criteria used to assess how innovative a project is.

## Source of Truth

Ontology attributes plus research on competitive landscape.

## Input

Ontology attributes and competitive research.

## Output

Reusable innovation-assessment criteria.

## Consumer

Reasoning and Valuation layers.

## Folder Structure

```
Innovation/
├── FirstMover.md      ├── CompetitiveMoat.md   ├── Execution.md
├── FastFollower.md    ├── ProductMarketFit.md  └── InnovationScore.md
├── Copycat.md         ├── Timing.md
└── CategoryCreator.md
```

## Workflow Position

Assessment sub-layer of the `Patterns` stage feeding `Reasoning` in `Research → Knowledge → Ontology → Patterns → Reasoning → Framework → Applications`.

## Rules

1. Define criteria only.
2. Criteria must generalise across projects.
3. No verdicts on real projects.

## Naming Convention

`PascalCase.md`.

## Future Expansion

Weighted and automated innovation scoring.
